❓ RiderVoice AI 사용 방법

🚀 처음 시작하기

1
서버 연결: 자동으로 설정됩니다. 연결 오류 시 대시보드 상단 "🔄 상태 확인"을 눌러 점검해보세요.
2
회원가입: 아이디/비밀번호 설정 → 개인정보 입력 → 배달 플랫폼 계정 연동(선택) 순으로 진행합니다.
3
로그인: 가입한 아이디와 비밀번호로 로그인합니다. 다음 방문 시 자동 로그인됩니다.

🔑 라이선스 관리

1
좌측 메뉴에서 라이선스를 선택하거나 대시보드의 "라이선스 등록" 버튼을 누르세요.
2
구매 시 받은 라이선스 키(LM1-XXXXXXXX-XXXXXXXX 형식)를 입력하고 등록합니다.
3
디바이스 ID는 자동으로 입력됩니다. 라이선스는 이 디바이스에 귀속됩니다.
💡 라이선스 일시정지가 필요하면 "쿠폰 적용" 탭에서 PAU1- 형식 쿠폰을 사용하세요.

🛵 배달 계정 연동

1
좌측 메뉴 배달 계정 → [+ 계정 연동] 버튼을 클릭합니다.
2
배달의민족 또는 쿠팡이츠 플랫폼을 선택 후 해당 아이디/비밀번호를 입력합니다.
3
연동된 계정은 목록에서 확인하거나 삭제할 수 있습니다.

🎫 쿠폰 사용

1
좌측 메뉴 쿠폰 탭에서 쿠폰 코드를 입력합니다. (PAU1- 형식: 일시정지 쿠폰)
2
디바이스 ID 확인 후 [쿠폰 적용] 버튼을 누르면 즉시 처리됩니다.

👤 내 정보 수정

1
이름, 휴대폰 번호 수정 후 저장 버튼을 클릭합니다.
2
비밀번호 변경 시 현재 비밀번호와 새 비밀번호(8자 이상)를 입력하고 저장합니다.
🔥 문제가 생기면 대시보드 상단 "🔄 상태 확인"으로 서버 연결을 점검하거나, ⚙ 서버 설정에서 URL을 다시 확인해보세요.
